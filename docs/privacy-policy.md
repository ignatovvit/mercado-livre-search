# ML Lens Privacy Policy

Effective date: 2026-07-13

ML Lens is a browser extension that provides advanced search, local query translation, filtering, scanning, and ranking for supported Mercado Livre and Mercado Libre marketplace pages.

## Data ML Lens Processes

ML Lens may process the following data locally in the user's browser:

- User-entered search queries.
- Translated search queries.
- Filter preferences.
- Panel position and interface settings.
- Marketplace listing content visible on supported pages.
- Product metadata extracted from supported marketplace pages, such as title, URL, price, rating, review count, seller signals, and material terms.
- Scan session state, including visited result pages and locally ranked products.

## How Data Is Used

ML Lens uses this data to:

- Translate Russian or English product queries into the current marketplace language.
- Open marketplace search pages.
- Filter and rank marketplace results.
- Continue user-initiated scan sessions across marketplace result pages.
- Restore user preferences across page reloads.

## Local Storage

ML Lens stores preferences and scan session state locally using browser extension storage. This data remains in the user's browser unless the user clears extension data or removes the extension.

## Network Requests

ML Lens may fetch product detail HTML from supported Mercado Livre and Mercado Libre marketplace domains when the user starts a scan. These requests are used to inspect product information for local filtering and ranking.

ML Lens does not send user data to a developer-operated server.

## Data Sharing

ML Lens does not sell, rent, or share user data.

ML Lens does not use analytics, advertising SDKs, tracking pixels, or behavioral profiling.

## Remote Code

ML Lens does not load or execute remote JavaScript. Extension scripts and dictionaries are packaged with the extension.

## Data Deletion

Users can delete ML Lens local data by clearing extension data in the browser or by removing the extension.
